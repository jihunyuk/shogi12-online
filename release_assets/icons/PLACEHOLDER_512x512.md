# App Icon Placeholder

## Required Specification

| Item | Value |
|------|-------|
| Size | 512 x 512 px |
| Format | PNG |
| Color space | sRGB |
| Transparency | Not allowed (Play Store) |
| Filename | icon-512.png |

## Naming Convention

| Target | Filename | Size |
|--------|----------|------|
| Play Store | icon-512.png | 512x512 |
| Adaptive icon foreground | icon-foreground.png | 108dp |
| Adaptive icon background | icon-background.png | 108dp |

## Design Notes

- No animal theme
- Strategy board game aesthetic
- Chinese character motif (王 / 將 / 相 / 子)
- Calm, minimal, serious tone
- Dark background recommended (#1a1a2e or similar)

## Placement

Place final icon assets in this directory:
`release_assets/icons/`
