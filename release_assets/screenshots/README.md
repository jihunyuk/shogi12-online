# Screenshot Guide

## Required Scenes

| Folder | Scene | Notes |
|--------|-------|-------|
| 01_main_menu | Main menu | Show all 5 options |
| 02_local_match | Local 2-player match | Mid-game state |
| 03_vs_computer | VS Computer match | AI move happening |
| 04_online_match | Online multiplayer | Both players connected |
| 05_rules_page | Rules page | Korean or English |
| 06_result_screen | Game result overlay | Win/lose state |

## Recommended Dimensions

- Phone: 1080 x 1920 px (portrait)
- Tablet: 1600 x 2560 px (portrait)

## Play Store Requirements

- Minimum 2 screenshots required
- Maximum 8 screenshots
- Format: PNG or JPEG
- No additional borders or device frames required by Play Store
