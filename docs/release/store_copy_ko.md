## 앱 제목

## 짧은 설명 (80자 이내)

4x3 전략 보드게임 — 대국, AI 대전, 온라인 멀티플레이 지원

## 긴 설명 (4000자 이내)

**쇼기12 온라인**은 전통 쇼기에서 영감을 받은 미니 전략 보드게임입니다.

### 게임 특징

- **4x3 소형 보드** — 빠르고 치열한 대국
- **30초 제한 턴** — 집중력과 순발력이 핵심
- **기물 재사용** — 잡은 기물을 아군 기물로 재배치
- **입왕 규칙** — 왕이 상대방 진영 끝까지 진입하면 승리 가능

### 게임 모드

- **로컬 대국** — 같은 기기에서 2인 대전
- **AI 대전** — 컴퓨터 상대와 연습 대국
- **온라인 멀티플레이** — 전 세계 플레이어와 실시간 대결

### 기물 구성

| 기물 | 이동 방식 |
|------|----------|
| 王 (왕) | 8방향 1칸 |
| 將 (장) | 상하좌우 1칸 |
| 相 (상) | 대각선 1칸 |
| 子 (자) | 전진 1칸 |
| 後 (후) | 승급한 子, 전방 5방향 |

### 승리 조건

1. 상대방 王을 잡으면 승리
2. 자신의 王이 상대방 마지막 줄에 도달하면 입왕 승리

한국어 / English
